# Gemini Project: Ops Site

This document provides a guide for interacting with the `Ops Site` project using Gemini.

## Project Overview

The `Ops Site` project is a collection of operational tools, scripts, and a website. It includes tools for monitoring flights, managing feeds, and network analysis.

## How to Use

To get started, you can ask Gemini to perform the following tasks:

*   **List files:** "List the files in the `6_ops-site` directory."
*   **Explore a subdirectory:** "List the files in the `flight_logger` directory."
*   **Read a file:** "Read the `www/index.html` file."
*   **Summarize a script:** "Summarize the `unified-feed-manager/install_all_feeds.sh` script."

*This document is intended for use with the Gemini CLI.*
